🔗 DATABASE SCHEMA
═══════════════════════════════════════════════════════════════════════════════

ai_financial_reports Table:
  • id (Primary Key)
  • user_id → Healthcare Professional
  • year → Fiscal year
  • report_type → Type of report
  • title, summary, analysis
  • recommendations → Generated by AI
  • key_metrics → JSON metrics
  • charts_data → JSON chart data
  • status → Lifecycle (draft→approved)
  • created_by, created_at, updated_at

ai_report_versions Table:
  • id (Primary Key)
  • report_id → Link to report
  • version_number → Version sequence
  • content → Version data
  • created_at → Timestamp

🌐 API ENDPOINTS

GET    /api/ai-reports.php?action=stats Get statistics
