Executive Summary

MediTax Connect is a SaaS platform that bridges the gap between healthcare professionals and certified accountants. The platform enables doctors, dentists, and pharmacies to upload financial documents, track income and expenses, and connect with verified accountants — all in one secure, role-based environment. The system supports three distinct user types: healthcare professionals (free access), accountants ($80/month subscription), and platform administrators.

Security Features
  • Password hashing with bcrypt (PASSWORD_DEFAULT)
  • CSRF token generation & validation
  • PHP Session-based authentication
  • Role-based access control (RBAC)
  • Input sanitization via htmlspecialchars
  • PDO prepared statements (SQL injection prevention)
  • File type validation on uploads
  • HIPAA-conscious data handling
